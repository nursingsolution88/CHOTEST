# NSO Deployment Guide

## Website Hosting
- Upload frontend files to hosting service.
- Connect Google Apps Script API URL.
- Configure Google Sheets database.

1. Create required Google Sheets.
2. Deploy Apps Script as Web App.
3. Add API URL in frontend configuration.
4. Test registration, exam, and result flow.

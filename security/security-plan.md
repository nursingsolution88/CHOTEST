# NSO Security Plan

## Student Data Protection
- Each student can access only their own profile and results.
- User identification through unique User ID.
- Sensitive data should be handled securely.

## Authentication
- Secure login validation.
- Session management.
- Admin access separation.

## Result Security
- Filter results using registered email/User ID.
- Prevent access to other student records.

## Google Sheets
- Use Apps Script permissions.
- Keep admin sheets protected.

# NSO Google Sheet Configuration

## Required Sheets

1. Users Sheet
- Name
- Email
- Registration Date
- User ID

2. Question Bank Sheet
- Test Name
- Subject
- Question
- Options A-D
- Correct Answer
- Explanation
- Test Timing

3. Results Sheet
- Student Name
- Email
- Test Name
- Score
- Percentage
- Date & Time

4. Test Settings Sheet
- Test Name
- Duration
- Status

Connect these sheets with Google Apps Script API for the NSO platform.
